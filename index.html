CSS Guide

CSS controls how your site looks — colors, sizes, fonts, spacing, and more. Use the "Custom CSS" button to type in any of these examples. Click an example to copy it!

Change Colors

body { background: #ffb6c1 !important; }
Changes the page background to light pink
nav { background: #e84393 !important; }
Makes the navigation bar pink
.section-title { color: #6c5ce7 !important; }
Makes all section titles purple
.hero { background: linear-gradient(135deg, #ff6b6b, #ffa502, #ff6348) !important; }
Changes the hero banner to a red-orange gradient

Change Fonts & Text

* { font-family: 'Comic Sans MS' !important; }
Changes ALL text to Comic Sans
.hero h1 { font-size: 8rem !important; }
Makes the main title really big
.section-title { text-transform: uppercase !important; letter-spacing: 5px; }
Makes all section titles UPPERCASE with spacing

Hide Things

.hero { display: none !important; }
Hides the hero section completely
nav { display: none !important; }
Hides the navigation bar
footer { display: none !important; }
Hides the footer

Fun Effects

.book-card:hover { transform: rotate(5deg) scale(1.1) !important; }
Makes book cards tilt and grow when you hover
.hero h1 { animation: popIn 1s ease infinite !important; }
Makes the title bounce forever
* { border: 2px solid red !important; }
Outlines every element in red (useful for seeing layout)
img { border-radius: 50% !important; }
Makes all images circular

Borders & Rounded Corners

.book-card { border: 3px solid var(--pink) !important; border-radius: 30px !important; }
Adds a pink border and extra round corners to book cards
nav { border-bottom: 4px solid var(--yellow) !important; }
Adds a yellow line under the nav bar

Spacing & Size

section { padding: 6rem 2rem !important; }
Adds more vertical space in each section
.book-cover { width: 250px !important; height: 375px !important; }
Makes book covers bigger

Shadows & Glow

.hero h1 { text-shadow: 0 0 20px #e84393, 0 0 40px #e84393, 0 0 80px #e84393 !important; }
Gives the title a pink neon glow
.book-card { box-shadow: 0 10px 40px rgba(232,67,147,0.4) !important; }
Adds a pink shadow behind book cards
Tips:
• Add !important at the end to override existing styles
• Colors can be names (pink, red) or codes (#ff69b4)
• Use var(--pink), var(--blue), var(--yellow), var(--green), var(--purple) for the site's own colors
• You can combine multiple rules — just put them all in the Custom CSS box
• Changes reset on refresh (or use Reset Page)

Heartstopper: Volume 1

Boy meets boy. Boys become friends. Boys fall in love. An LGBTQ+ graphic novel about life, love, and everything that happens in between. Nick and Charlie are at the same school, but they've never met — until one day when they're made to sit together. They quickly become friends, and soon Charlie is falling hard for Nick, even though he doesn't think he has a chance.
